####Augmented Reality Dance Game using Kinect.
Augmented Reality Dance Game using Kinect is expected as one of dance game innovation. The previous famous dance game known as dance dance revolution that using dance pad is using virtual reality concept, and the following is just dance game in Xbox that using kinect and mixed reality / augmented virtuality concept. This game offers augmented reality concept with more feature. Unreal object will be shown to real world. Players will feel as they are at stage and in front of audience. Inside game development, game needs to calculating length of human hands and foots, starts from shoulder until its hand flat, or from hips until foot flat and adjusted to screen resolution. With some major problem at game development, there is some conclusion about how to measuring the length of human hands from it shoulder until hand flat and human foots from it hips until soles of the feet, using video frame and body detection with kinect SDK, human hands elevation angle, and organize a scene inside game development.
